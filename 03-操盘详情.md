# 操盘详情 · 运营功能说明(v1 现状版)

> **文档定位**:本文档面向运营人员,逐个说明【操盘详情】模块中每个按钮的功能用途、操作步骤、操作后果、权限要求与注意事项。
>
> **对应版本**:基于 GitHub master 分支(2026-05-25)`操盘详情/` 16 章完整文档
>
> **使用方法**:按位置分组导航——顶部栏 → 赛事信息头 → 左侧切场面板 → 盘口工具栏 → 盘口卡片 → 右侧监控面板 → 各类弹窗。每个按钮均含【按钮名 / 位置 / 用途 / 操作步骤 / 操作后果 / 权限 / 注意事项】7 项说明。

---

### 模块定位

操盘详情是操盘手对**单个赛事**的工作台,提供:
- 赛事级控制(整体上架/下架、锁定、隐藏、数据源跟随)
- 盘口级操作(单盘口的赔率/状态调整)
- 风控监控(风险注单 + 最近投注 + 告警)
- 切场快捷(快速切换到其他赛事)

**核心角色**(本模块):
- **操盘手**:管理自己负责的赛事盘口,可改赔率、控制状态
- **主管**:全组操盘 + 人员分配
- **风控**:全局监控 + 紧急锁定 + 风控参数配置

## 二、顶部栏 — 中间:盘口批量操作组

> 顶部第一组按钮组,**互斥显示设计**(隐藏/取消隐藏 一组,锁定/解锁 一组)。

### 3. [⏸ 全部隐藏] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 全部隐藏 |
| **位置** | 顶部栏盘口批量操作组,左侧 |
| **用途** | 将当前赛事所有非关盘盘口状态设为"隐藏"(玩家不可见) |
| **操作步骤** | ① 点击 [⏸ 全部隐藏] → ② 弹出确认弹窗 → ③ 点 [确认] |
| **操作后果** | 当前赛事所有非关盘盘口变为"隐藏";按钮切换为 [▶ 取消隐藏];操盘日志记录 |
| **权限** | 操盘手(限自己负责的赛事)/ 主管 / 风控 |
| **注意事项** | **与 4 [取消隐藏] 互斥**,只显示一个;**锁定状态下此按钮禁用**(灰色),需先解锁;只影响"非关盘"状态的盘口 |

### 4. [▶ 取消隐藏] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 取消隐藏 |
| **位置** | 顶部栏盘口批量操作组(替代 3 [全部隐藏] 显示) |
| **用途** | 将赛事所有隐藏盘口恢复为"开盘"状态 |
| **操作步骤** | ① 点击 [▶ 取消隐藏] → ② 弹出确认弹窗 → ③ 点 [确认] |
| **操作后果** | 所有隐藏盘口恢复开盘;按钮切换为 [⏸ 全部隐藏];玩家恢复可见 |
| **权限** | 操盘手(限自己负责的赛事)/ 主管 / 风控 |
| **注意事项** | **仅在赛事处于"隐藏"状态时显示**;恢复后立即生效,客户端可见 |

### 5. [🔒 全部锁定] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 全部锁定 |
| **位置** | 顶部栏盘口批量操作组,右侧 |
| **用途** | 将赛事所有非关盘盘口设为"锁定"(玩家可见但暂停投注) |
| **操作步骤** | ① 点击 [🔒 全部锁定] → ② 弹出锁定确认弹窗 → ③ 填写锁定原因 → ④ 确认 |
| **操作后果** | 所有非关盘盘口变为锁定;按钮切换为 [🔓 解锁];客户端可见但无法投注;操盘日志记录 |
| **权限** | 操盘手(限自己负责的赛事)/ 主管 / 风控 |
| **注意事项** | **危险按钮**(红色边框);锁定 ≠ 隐藏:锁定时玩家**仍可看到盘口但不可投注**;隐藏时玩家**完全看不到** |

### 6. [🔓 解锁] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 解锁 |
| **位置** | 顶部栏盘口批量操作组(替代 5 [全部锁定] 显示) |
| **用途** | 将赛事所有锁定盘口恢复为锁定前的状态 |
| **操作步骤** | ① 点击 [🔓 解锁] → ② 弹出确认弹窗 → ③ 确认 |
| **操作后果** | 锁定盘口恢复;按钮切换为 [🔒 全部锁定];客户端可恢复投注 |
| **权限** | 操盘手(限自己负责的赛事)/ 主管 / 风控 |
| **注意事项** | **仅在赛事处于"锁定"状态时显示**;恢复到的是"锁定前的状态"(不一定是开盘),如锁定前是隐藏则恢复为隐藏 |

## 三、顶部栏 — 中间:赛事操作组

| 项 | 内容 |
|---|---|
| **按钮名** | 下架 / 上架(互斥显示) |
| **位置** | 顶部栏赛事操作组,上架状态标签右侧 |
| **用途** | 下架:把整个赛事从客户端撤下;上架:将赛事重新发布到客户端 |

| **注意事项** | **已上架显示下架按钮**,**已下架显示上架按钮**;上架时**必须选择盘口初始状态**(跟随数据源 / 锁定 / 隐藏) |

## 📋 常见操作场景速查

### 场景 1:进入赛事后调整赔率

```
步骤:
  ① 从操盘列表点击行进入操盘详情
  ② 在中间盘口卡片区找到要调整的盘口
  ③ 点击赔率单元格 29,进入编辑
  ④ 输入新赔率,按 Enter 确认
  ⑤ 弹出赔率变更确认弹窗(52),点 [确认变更]
  ⑥ 赔率保存到本地,顶部栏会提示"需同步市场"
  ⑦ 点 [📈 同步市场 11] 推送到客户端
  ⑧ (可选)点 [📋 操盘日志 10] 确认操作已记录

注意:
  - 调赔率会自动关闭 [数据源跟随开关 9]
  - 超出 HK/RTP/调幅边界会弹出校验弹窗(50/51)
```

### 场景 2:发现单边告警,紧急锁定相关盘口

```
步骤:
  ① 顶部栏 [告警摘要 12] 看到"⚠️ 单边:3"
  ② 点击告警摘要,系统定位到对应盘口
  ③ 单盘口锁定:点击该盘口的 [盘口状态控制 28] → [锁定]
  ④ 或整赛事锁定:点击 [🔒 全部锁定 5] → 弹出锁定确认弹窗
  ⑤ 填写锁定原因 [39] → 点 [确认锁定 40]
  ⑥ 处理完异常后,用 [🔓 解锁 6] 恢复
```

### 场景 3:切换到其他赛事(有未保存编辑)

```
步骤:
  ① 左侧切场面板找目标赛事
  ② 点击赛事卡片 20
  ③ 系统检测到当前赛事有未保存编辑
  ④ 弹出切场确认弹窗 49
  ⑤ 选 [放弃修改]:跳转目标赛事(修改丢失)
     或 [留下继续编辑]:保存好再切

注意:
  - 切场前建议先 [📈 同步市场 11] 把编辑同步出去
```

### 场景 4:赛事结束前下架

```
步骤:
  ① 顶部栏点 [⬇️ 下架 7]
  ② 弹出下架确认弹窗 41
  ③ (可选)填写下架备注 42
  ④ 点 [确认下架]
  ⑤ 整个赛事变为已下架,客户端不可见
```

### 场景 5:风控人员调整告警阈值

```
步骤:
  ① 点 [⚙️ 设置 15] → 选择"风控设置"
  ② 弹出风控设置弹窗
  ③ 调整对应参数(告警阈值/偏离阈值等)55
  ④ 点 [保存 56]
  ⑤ 参数立即生效

注意:
  - 仅风控可改
  - 不当配置可能导致告警风暴或漏报
```

## 📌 注意事项汇总

```
🔴 高危操作(影响面广):
   - [🔒 全部锁定 5] → 整赛事锁定,影响所有盘口
   - [⬇️ 下架 7] → 整赛事撤下,客户端不可见
   - [赔率单元格 29] → 调赔率会自动关闭数据源跟随
   - [强制通过校验 51] → 跳过赔率校验提交,需有权限,会审计
   - [风控参数 55] → 影响告警敏感度

🟡 必填项 / 操作前提:
   - [上架确认 44] 必须选初始盘口状态
   - [锁定确认 40] 建议填原因
   - [赔率变更 52] 后需要 [同步市场 11] 才能推到客户端

🟢 仅查看,不可操作:
   - [告警摘要 12]、[数据源状态 13]、[赔率趋势 32]、[投注分布条 33]

🔵 角色权限差异:
   - 普通操盘手:仅操作自己负责的赛事,部分按钮如 [强制通过校验 51] 无法点击
   - 主管:全组操盘 + 人员分配 [48]
   - 风控:跨赛事 + 风控参数配置 [55]

⚠️ 操作不可一键撤销:
   - 所有"确认"按钮都直接生效
   - 调错赔率/锁定/下架后,只能再操作一次回退
   - 操盘日志可追溯,但不能 undo

⚠️ 数据同步:
   - 本地赔率编辑 ≠ 客户端可见,需 [📈 同步市场 11]
   - WebSocket 自动推送行情,异常时 [🔄 刷新 14]
```
