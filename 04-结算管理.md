# 结算管理 · 运营功能说明(v1 现状版)

> **文档定位**:本文档面向运营人员(结算人员/结算主管/风控人员/财务人员),逐个说明【结算管理】模块中每个按钮的功能用途、操作步骤、操作后果、权限要求与注意事项。
>
> **对应版本**:基于 GitHub master 分支(2026-05-25)`结算管理/` 19 章完整文档
>
> **使用方法**:按位置分组导航——左侧筛选栏 → 顶部工具栏 → 数据列表(赛事视图/注单视图) → 批量操作栏 → 分页 → 各类弹窗。每个按钮均含【按钮名 / 位置 / 用途 / 操作步骤 / 操作后果 / 权限 / 注意事项】7 项说明。

---

## 模块定位

结算管理是结算人员的**主战场**,展示所有待处理的结算赛事和注单。功能包括:
- 赛事筛选(按状态 / 阶段 / 运动类型)
- 赛事视图 + 注单视图双模式
- 注单结算(L3/L4 手工选结果 / L2 系统预填确认)
- 异常赛事处理(取消 / 延期 / 腰斩 / 数据错误)
- 批量分配结算人员

**核心角色**:
- **结算人员**:一线结算操作(选结果、发起盘口二次结算、处理异常)
- **结算主管**:结算人员全部 + 人员分配 / 批量分配
- **风控人员**:查看异常赛事、风控告警(只读)
- **财务人员**:查看数据(只读)

| 项 | 内容 |

| **按钮名** | 全部赛事 |
| **位置** | 左侧筛选栏「赛事结算状态」组,首选项 |

# 📋 常见操作场景速查

## 场景 1:开班——查看待结算赛事

```
步骤:
  ① 默认进入"赛事视图 9"
  ② 左侧筛选选 [待结算赛事 2]
  ③ 顶部 [时间快捷"今天" 13] 或 [自定义时间 11]
  ④ 列表显示所有今天待结算的赛事
  ⑤ 选 [我负责的](如有该筛选)或筛 [结算人员 14] = 自己
```

## 场景 2:逐注单结算(L3/L4)

```
步骤(结算人员):
  ① 切换到 [注单视图 9]
  ② 筛选 [投注阶段 17] + [结算人员 14] = 自己
  ③ 找到 L3/L4 注单(蓝色"结算"按钮 29)
  ④ 点 [结算] → 弹出注单结算弹窗
  ⑤ 选择 [结算结果 39] → 点 [确认结算 41]
  ⑥ 二次确认 → 提交
  ⑦ 注单状态变"已结算"

注意:
  - 同盘口同基准分组的其他注单会联动结算
  - 关联串关母单可能立即触发短路结算
```

## 场景 3:结算主管批量分配人员

```
步骤(主管):
  ① [赛事视图 9],按时间/联赛筛选目标赛事
  ② 表头复选框 20 全选,或点 [全选 N 场 21] 跨页全选
  ③ 底部出现操作栏,点 [分配结算人员 33]
  ④ 弹出分配弹窗
  ⑤ 切换 [赛前/滚球 Tab 44]
  ⑥ 选 [结算人员 45]
  ⑦ 两阶段都设好 → 点 [确认分配 46]
  
注意:
  - 异常赛事仍可分配(需有人处理)
  - 已结算赛事按钮禁用(无任务)
```

## 场景 4:处理异常赛事(腰斩暂缓)

```
步骤(结算人员):
  ① 左侧 [异常赛事 4] 筛选
  ② 找到要处理的异常赛事 → 点 [详情 25] 跳转详情页
  ③ 详情页"赛事异常处理"按钮已隐藏(因为已是异常态)
  ④ 在详情页先点 [赛事驳回] 回到"待结算"
  ⑤ 再次进入异常处理流程(在详情页执行)

注意:
  - 操作都在结算详情页完成,结算列表只展示和跳转
  - 异常处理 4 种:取消 / 延期 / 腰斩 / 数据错误
```

## 场景 5:已结算赛事在 24h 内需要驳回

```
步骤:
  ① 左侧 [已结算赛事 3] 筛选
  ② 找到要驳回的赛事 → 点 [详情 25]
  ③ 在详情页点 [赛事驳回] 按钮(详情页第 15.3 节)
  ④ 赛事回到"待结算"状态,在结算列表重新出现
  ⑤ 重新发起结算流程

注意:
  - 24h 窗口很关键,超过就不能驳回
  - 驳回赛事不等于驳回注单,**注单回退需配合"盘口二次结算"**
```

# 📌 注意事项汇总

```
🔴 高危操作:
   - [批量分配结算人员 33] → 影响多个赛事人员归属
   - [跨页全选 21] → 超 50 场会二次确认
   - [赛事驳回] → 详情页执行,影响已结算赛事

🟡 必填 / 关键:
   - [注单结算 41] 必经二次确认
   - [显示未完成 12] 建议保持勾选(避免漏单)
   - 异常处理流程:必须先驳回再异常

🟢 仅查看 / 跳转:
   - [赛事行点击 26]、[详情 25] 都是跳转入口
   - 风控/财务全部只读

🔵 角色权限差异:
   - 结算人员:做注单结算 + 处理异常
   - 结算主管:全部 + 人员分配 [33]
   - 风控人员:异常赛事审查(只读,无操作按钮)
   - 财务人员:数据查看(只读,无操作按钮)
   - **风控/财务无批量操作栏可见**

⚠️ 视图差异:
   - 赛事视图有批量操作栏
   - 注单视图无批量操作
   - 视图切换 9 保留筛选条件

⚠️ 状态约束:
   - 已结算赛事 24h 内可驳回,超时归档
   - 异常态不能直接异常处理,需先驳回回待结算
   - 同基准分组注单结算会联动(可能影响其他注单)
```
