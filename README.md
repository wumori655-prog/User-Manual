## 👥 角色与权限速查

| 角色 | 主要工作模块 | 核心操作 |
|---|---|---|
| **操盘手(普通)** | 操盘列表、操盘详情 | 调赔率、上架/下架自己负责的赛事、锁定盘口 |
| **操盘主管** | 操盘列表、操盘详情 | 全组操盘 + 人员分配 + 紧急锁盘 |
| **结算人员** | 结算管理、结算详情 | 赛果录入、注单结算、异常处理、盘口二次结算 |
| **结算主管** | 结算管理、结算详情 | 一线全部 + 结算人员分配 / 批量分配 |
| **风控人员** | 各模块(只读) | 异常赛事审查、风控告警查看 |
| **财务人员** | 各模块(只读) | 数据查看 |
| **运营管理人员** | 联赛管理 | 联赛配置、操盘手分配 |

> **角色对齐原则**:同一自然人可承担多个岗位,按操作页面识别身份(操盘页面=操盘手,结算页面=结算人员)。

## 📌 重要约定

```
🔴 高危操作 — 影响面广,操作前请二次确认:
   - 联赛等级变更、玩法全部禁用、批量下架、一键锁盘
   - 盘口二次结算、赛事异常处理、强制通过校验

🟡 立即生效 — 操作即时影响线上:
   - 跟随数据源开关、单盘口状态变更
   - 任何已保存的配置变更
   
🟢 仅未来生效 — 不影响已上架赛事:
   - 默认操盘手变更、联赛等级变更、玩法启用变更
   
🔵 操作不可一键回滚 — 改错只能再改回
```
