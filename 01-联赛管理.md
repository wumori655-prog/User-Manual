## 模块定位

联赛管理是操盘系统的**配置中心**,用于管理联赛级别的操盘参数。配置变更会影响该联赛下所有赛事的行为。每个联赛包含 7 项核心配置:

| 配置项 | 默认值 | 影响范围 |
|---|---|---|
| 联赛状态 | 开启 | 决定赛事是否可上架 |
| 联赛等级 | 默认档 | 关联风控管理的限额配置(默认 + 等级 1-10 共 11 档) |
| 跟随数据源盘口状态 | 是 | 控制 IM 暂停状态是否影响 C 端展示 |
| 默认操盘手 | 空 | 赛前/滚球分别配置,批量上架时使用 |
| 串关支持 | 是 | 该联赛赛事是否支持串关 |
| 玩法启用列表 | 全部启用 | 一期足球 208 种 FT 玩法(可操盘 82 种 + 不可操盘 126 种) |
| 主力线显示规则 | 仅显示主力 | 控制副线是否默认展示 |

# 三、列表行操作按钮(每行 2 个)

> 每行末尾"操作"列含 2 个按钮:**⑧ [配置]** 和 **⑨ [操盘手]**。

## ⑧ [配置] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 配置 |
| **位置** | 列表每行末尾"操作"列,左侧蓝色主按钮 |
| **用途** | 打开该联赛的核心配置弹窗,编辑联赛等级、跟随数据源、串关支持、玩法启用等核心配置(默认操盘手在另一个按钮 ⑨) |
| **操作步骤** | ① 在列表中找到要配置的联赛 → ② 点击该行的 [配置] 按钮 → ③ 配置弹窗打开,显示当前配置 → ④ 修改需要的项 → ⑤ 点击 ⑲ [保存配置] 提交 |
| **操作后果** | 弹窗打开,显示该联赛的现有配置;修改后保存,配置变更按生效规则(见下方表格)应用 |
| **权限** | 主管 / 运营管理人员可改;一般操作员仅可查看 |
| **注意事项** | 默认操盘手**不在此弹窗里**,在 ⑨ [操盘手] 按钮的弹窗里;**配置变更对已上架赛事的影响**有明确规则(见下方说明) |

### 配置变更生效规则(重要)

| 变更项 | 对已上架赛事 | 对待上架赛事 |
|---|---|---|
| 联赛禁用 | 不影响 | 从列表中移除 |
| 联赛等级变更 | 不影响(保持原等级限额) | 新上架时使用新等级限额 |
| 跟随配置变更 | **立即生效**(例外,见操盘列表第 16 章) | 按新配置生效 |
| 默认操盘手变更 | 不影响 | 新上架时使用新配置 |

## ⑨ [操盘手] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 操盘手 |
| **位置** | 列表每行末尾"操作"列,[配置] 按钮右侧的灰色次按钮 |
| **用途** | 打开默认操盘手分配弹窗,为该联赛的【赛前 / 滚球】两个阶段分别按玩法分配默认操盘手 |
| **操作步骤** | ① 在列表中找到要分配的联赛 → ② 点击该行的 [操盘手] 按钮 → ③ 弹窗打开,默认显示"赛前阶段" Tab → ④ 在左侧"可分配玩法"中选择玩法 → ⑤ 在中间下拉框选择操盘手 → ⑥ 点击 [分配 →] → ⑦ 切换到"滚球阶段" Tab 重复 ④~⑥ → ⑧ 点击 [保存设置] |
| **操作后果** | 该联赛的默认操盘手配置更新;变更**仅对新上架赛事生效**,已上架赛事不受影响 |
| **权限** | 主管 / 运营管理人员可改 |
| **注意事项** | **赛前 / 滚球阶段需要分别配置**;玩法按 6 组管理(让球 / 大小 / 角球 / 进球 / 半场 / 特殊);批量上架时选择"按联赛默认"会使用此配置 |

| **操作步骤** | ① 点击 [📅 赛前阶段] 或 [🔴 滚球阶段] Tab → ② 弹窗显示该阶段对应的玩法分配状态 |
| **操作后果** | Tab 切换;两个阶段的分配数据**完全独立**,互不影响 |
| **权限** | 主管 / 运营管理人员 |
| **注意事项** | 默认显示"赛前阶段";**必须分别配置两个阶段**才能覆盖完整赛事生命周期;切换 Tab 不丢失当前阶段未保存的修改 |

## ㉒ [让球/大小/角球/进球/半场/特殊] 6 组快捷选择按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 6 组快捷选择(让球 / 大小 / 角球 / 进球 / 半场 / 特殊) |
| **位置** | 操盘手弹窗左侧"可分配玩法"面板底部 |
| **用途** | 快速勾选某一组的全部玩法,免去逐个点击 |
| **操作步骤** | ① 点击想要快速选择的组(如 [让球]) → ② 该组所有玩法自动勾选 → ③ 配合中间的操盘手下拉一起分配 |
| **操作后果** | 该组全部玩法在"可分配玩法"列表中被勾选;再次点击同一组按钮不会取消(用 ㉔ [清空] 才能取消) |
| **权限** | 主管 / 运营管理人员 |
| **注意事项** | 6 组对应风控管理的 6 玩法分组;某些组(如"进球")玩法很多(100 种),建议用快捷按钮一次性选 |

## ㉖ [分配 →] 按钮

| 项 | 内容 |
|---|---|
| **按钮名** | 分配 → |
| **位置** | 操盘手弹窗中间,[选择操盘手] 下拉下方 |

| **权限** | 主管 / 运营管理人员 |
| **注意事项** | **必须同时:左侧有勾选 + 下拉有选操盘手**,否则不能分配;同一玩法可以**重新分配**(覆盖原操盘手) |

# 📋 常见操作场景速查

## 场景 1:新联赛上线,配置等级和操盘手

```
步骤:
  ① 用 ② 搜索 或 ③④⑤⑥⑦ 筛选找到该联赛
  ② 点 ⑧ [配置] → 设置联赛等级 ⑬、跟随数据源 ⑭、串关支持 ⑮、玩法启用 ⑰、主力线规则 ⑯
  ③ 点 ⑲ [保存配置]
  ④ 再点 ⑨ [操盘手] → 切换 ㉑ "赛前阶段" Tab → 用 ㉒/㉓ 选玩法 → ㉕ 选操盘手 → ㉖ [分配 →]
  ⑤ 切换 ㉑ "滚球阶段" Tab → 重复 ④
  ⑥ 点 ㉙ [保存设置]
```

## 场景 2:某联赛 IM 频繁误推暂停,需要忽略

```
步骤:
  ① 找到该联赛 → 点 ⑧ [配置]
  ② 关闭 ⑭ [跟随数据源盘口状态] 开关
  ③ 点 ⑲ [保存配置]
  → 立即生效,该联赛之后不再因 IM 暂停而显示 C 端暂停投注
```

## 场景 3:操盘手离职,需要把他的联赛转给别人

```
步骤(逐个联赛操作):
  ① 找到该操盘手负责的联赛 → 点 ⑨ [操盘手]
  ② 在右侧"已分配玩法"勾选离职操盘手的所有条目
  ③ 点 ㉗ [← 移除]
  ④ 在左侧重新勾选这些玩法
  ⑤ 用 ㉕ 选新操盘手
  ⑥ 点 ㉖ [分配 →]
  ⑦ ㉑ Tab 两个阶段(赛前 + 滚球)都要做
  ⑧ 点 ㉙ [保存设置]

注意:v1 不支持跨联赛批量改人;需要逐个联赛重复
```

## 场景 4:暂时停用某联赛(临时不允许投注)

```
步骤:
  ① 找到该联赛 → 点 ⑧ [配置]
  ② 联赛状态改为"暂停"或"关盘"
  ③ 点 ⑲ [保存配置]
  → 已上架赛事不影响,但新赛事不会进入待上架列表
```

# 📌 注意事项汇总

```
🔴 高危操作(影响面广):
   - 联赛等级 ⑬ 变更 → 影响该联赛所有未来赛事的风控限额
   - 玩法 ⑰ 全部禁用同一组 → 该联赛失去该类玩法,影响 C 端体验
   - 联赛禁用 → 该联赛新赛事不会出现

🟡 立即生效(注意时机):
   - 跟随数据源 ⑭ 开关变更 → 已上架赛事立即受影响

🟢 仅未来生效:
   - 默认操盘手 ㉙ 变更 → 仅新上架赛事使用新人员
   - 联赛等级 ⑬ 变更 → 已上架赛事保持原限额
   - 玩法 ⑰ 启用变更 → 仅影响新上架赛事的玩法

🔵 操作不可一键回滚:
   - 所有保存都直接生效,改错只能再改回
   - 建议大改动前先记录原配置
```
